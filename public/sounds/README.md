# Instructions pour le Sound Design

Veuillez déposer vos fichiers sons dans ce dossier :

| Fichier | Description | Statut |
|---------|-------------|--------|
| `startup.mp3` | Son d'ambiance futuriste à l'ouverture du site | ⏳ À ajouter |
| `click.mp3` | Son digital court pour les clics sur boutons | ✅ Présent |
| `success.mp3` | Son harmonieux pour les validations de formulaire | ⏳ À ajouter |

## Déclencheurs

- **Startup** : Se joue à la fin de l'animation du Preloader
- **Click** : Se joue sur les boutons CTA et liens principaux
- **Success** : Se joue lors de la validation réussie d'un formulaire

## Notes techniques

- Volume recommandé : 20-30%
- Les sons s'activent après le premier clic (règles navigateurs)
- Si un fichier est absent, le site continue sans erreur
